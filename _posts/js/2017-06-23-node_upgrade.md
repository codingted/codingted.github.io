# 升级nodejs

* 检查当前nodejs版本
```
node -v
```

* 清空npmcatch
```
sudo npm cache clean -f
```

* 安装“n”
```
sudo npm install -g n
```

* 安装指定版本
```
sudo n 0.8.11
或
sudo n stable
```

* 检查你的版本
```
node -v
```
